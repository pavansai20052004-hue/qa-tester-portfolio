# QA Tester Portfolio

Personal portfolio for GJV Pavan Sai, built for QA testing roles.

## Stack

- React 19
- Vite 8
- CSS

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Highlights

- Resume-backed QA portfolio content
- Recruiter-focused hero and project case studies
- Responsive layout for desktop and mobile
- Downloadable resume included in `public/`
